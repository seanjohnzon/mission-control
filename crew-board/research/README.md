# Robin's Research Index
This is the crew's knowledge base. Robin maintains it. Everyone reads it.

## Structure
- index.md — master index of all research, categorized with links
- improvements.md — actionable improvements found, scored by impact
- tech-stack.md — research relevant to current tech (R3F, Three.js, OpenClaw, etc.)
- tools.md — tools, frameworks, libraries the crew should know about
- security.md — security advisories, best practices, vulnerabilities
- trends.md — industry trends, AI agent developments, competitive intel
- per-agent/ — research specifically relevant to each agent's role

## Rules
- Robin APPENDS new findings, never overwrites
- Every entry has: [DATE] Title — 1-line summary — link to source
- Improvements have impact score (1-10) from Usopp
- Each agent checks their per-agent file on every cron cycle
