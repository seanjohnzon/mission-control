## QA Patterns and Testing Strategies

[2026-03-19] OWASP Testing Guide v4 — A comprehensive framework for web application penetration testing, offering best practices and detailed techniques for identifying security issues. Includes new testing methods like HTTP Verb Tampering and HTTP Parameter Pollution. — [OWASP Testing Guide](https://www.owasp.org/index.php/OWASP_Testing_Guide_v4_Table_of_Contents)

[2026-03-19] ISTQB Certified Tester Foundation Level Agile Tester (CTFL-AT) — Provides key testing skills for Agile projects, aligning with Agile Manifesto principles. Focuses on collaboration, adapting testing to Agile values, and supporting test automation. — [ISTQB CTFL-AT](https://istqb.org/certifications/certified-tester-foundation-level-agile-tester-ctfl-at/)

## Security Testing Resources

[2026-03-19] National Vulnerability Database (NVD) Monitoring — Essential for staying updated on security advisories and vulnerabilities in web applications. Regularly check NVD for the latest CVEs related to technologies in use to prioritize security testing efforts. — [NVD](https://nvd.nist.gov/vuln/search)
