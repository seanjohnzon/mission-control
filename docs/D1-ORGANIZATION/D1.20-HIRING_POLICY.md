# D1.20 — Hiring Policy

No agent joins the crew without proper documentation and scope verification.

## Tiers
- **Dept Leader**: Requires Captain approval. Enhanced hiring process (D2.1).
- **Sub-agent**: Dept leader hires within their budget. Standard hiring process.

## Access Control
| Doc Folder | Can Write | Can Read |
|-----------|-----------|----------|
| D1-ORGANIZATION | Nami + Captain | ALL |
| D2-RUNBOOKS | Nami + Usopp | ALL |
| D3-TEMPLATES | Usopp | ALL |
| D4-DEPARTMENT/X | Dept X leader | Dept X only |
| D5-REFERENCE | Nami | ALL |

New hires read policies but CANNOT edit them.
Dept leaders manage ONLY their own D4 folder.

## Model Budget
- Subscription models ($0): Opus, Sonnet, GPT-5.4, Haiku — prefer these
- Variable cost: Grok, Gemini — only for specialized roles
- Anthropic: Nami + Zoro departments ONLY (budget conservation)
- OpenAI: Everyone else
